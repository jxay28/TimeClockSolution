# T4.1 - Parametri globali in cartella dati condivisa

## Obiettivo

Usare `parametri_straordinari.json` dentro la cartella dati selezionata (`_csvFolder`), evitando dipendenza dalla working directory.

## Implementazione

- `App.ParametriFilePath` gestisce il path effettivo del file parametri.
- `App.ConfigureDataFolder(...)`:
  - risolve il path in base a `CsvFolderPath`;
  - crea il file se manca;
  - carica i parametri dal nuovo path;
  - copia i parametri dal vecchio path se necessario (prima migrazione).
- `App.SalvaParametriGlobali()` salva sempre su `ParametriFilePath`.
- `MainWindow` richiama `App.ConfigureDataFolder(...)`:
  - in avvio;
  - quando l'utente cambia cartella dati.

## Risultato

Configurazione coerente anche cambiando working directory o avviando l'app da percorsi diversi.
